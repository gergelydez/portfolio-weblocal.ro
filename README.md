# 🚀 Portfolio Demo — 3 Site-uri Ready to Deploy

## Deploy pe Vercel în 3 minute

```bash
npm install
npm run dev     # http://localhost:3000
```

### Deploy pe Vercel (gratuit):
1. Push pe GitHub
2. Mergi pe vercel.com → "Import Project"
3. Selectează repo-ul → Deploy
4. Ai linkul live în 2 minute

## Personalizare rapidă pentru client

Când ai un client, modifici în fișierul paginii:
- Numele afacerii
- Numărul de telefon
- Adresa
- Prețurile serviciilor
- Culorile (variabilele de culoare sunt în header-ul fiecărui component)

Deploy nou pe Vercel = link live în 60 secunde.
